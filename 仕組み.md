# 仕組み
簡単なプログラムの解説です。  

## y-tグラフ生成関数
時間`t`を渡すと、`x=0`の時の`y`を返す関数`f`   
`f(t)=hoge`  
を定義します。  
ex: `f(t)=cos(t)`  

## 波生成関数
定数を表にまとめます  

|定数|解説|
|-|-|
|w|装置の幅|
|v|波の速度|
|e|反射。固定端/自由端/反射しないのどれか|

時間`t`と座標`x`を渡すと座標`y`を返す関数`p`です。  
`y1`は通常の波、`y2`は反射波です。  

```
p(t,x,y)=
  y1=f(t-x/v)
  y2=f(t-(x+(w-x)*2)/v)
  eが自由端の時:
    y1+y2
  eが固定端の時:
    y1-y2
  eが反射しないの時:
    y1
```
